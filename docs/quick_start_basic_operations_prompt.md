# Resophy 基本操作功能 Prompt

请生成一个展示 Resophy 基本操作功能的示意图，包含以下核心功能模块：

## 1. 📚 论文管理

**核心功能**：
- **上传论文**：
  - 拖拽 PDF 文件上传
  - 输入 arXiv URL 直接下载论文
- **分类管理**：
  - 左侧分类树组织论文
  - 创建、重命名、删除分类
  - 论文移动到不同分类
- **全文搜索**：
  - 顶部搜索框
  - 支持标题、作者、摘要等字段搜索
  - 实时搜索结果

**界面元素**：
- 左侧分类树导航
- 中间论文列表（卡片式展示）
- 顶部搜索栏
- 上传按钮/拖拽区域

---

## 2. 📖 论文查看与阅读

**功能**：
- 点击论文卡片进入详情页
- PDF 阅读器查看论文
- 阅读时间自动记录
- 支持全屏阅读模式

**操作流程**：
1. 在论文列表中点击论文卡片
2. 进入论文详情页
3. 查看论文元数据（标题、作者、摘要等）
4. 点击 "查看 PDF" 打开阅读器
5. 系统自动记录阅读时间

**界面元素**：
- 论文详情页
- PDF 阅读器（支持缩放、翻页）
- 阅读时间统计

---

## 3. ✏️ 元数据管理

**功能**：
- 编辑论文信息
- 添加备注和标签
- 管理 BibTeX 引用
- 设置链接（GitHub、主页）

**可编辑字段**：
- 标题、作者、机构
- 年份、期刊、摘要
- 关键词、标签
- 用户备注
- BibTeX 引用
- GitHub、主页链接
- 收藏标记

**操作流程**：
1. 进入论文详情页
2. 点击 "编辑" 按钮
3. 修改论文信息
4. 保存更改

**界面元素**：
- 论文详情编辑面板
- 表单输入框
- 保存按钮

---

## 4. 📈 阅读历史

**功能**：
- 自动记录每篇论文的阅读时间
- 生成阅读热力图（可视化每日阅读统计）
- 支持不同颜色主题
- 查看最近阅读记录

**界面元素**：
- 阅读热力图（日历视图，不同颜色表示阅读时长）
- 最近阅读列表
- 颜色主题选择器

---

## 5. 📥 导出功能

**功能**：
- 导出论文库为 JSON 格式
- 包含所有论文元数据和分类结构
- 便于数据迁移和多平台使用

**操作流程**：
1. 进入设置界面
2. 找到导出功能
3. 选择导出范围（全部或指定分类）
4. 下载 JSON 文件

**界面元素**：
- 导出按钮
- 导出选项选择
- 下载提示
