1
准备导出文件

系统将自动解析 ZIP 文件并导入所有论文、目录结构、翻译和解读。导入过程可能需要一些时间。

4
查看结果

导入完成后，可以在主界面查看导入的论文和目录结构。

欢迎使用 Resophy

1. 配置 AI 功能

前往 Settings → Agentic 设置 LLM API 和 MinerU 服务，这是使用 AI 翻译，AI 解读，Daily Arxiv 的前提。

2. 基本操作

上传 PDF 论文、创建分类管理、使用搜索功能查找论文。点击论文可查看详情，双击打开阅读器。

3. Agentic Tools

支持 AI 解读 和 AI 翻译 功能。AI 解读可将论文转换为详细的 Markdown 格式解读文章，AI 翻译可生成中英双语 PDF。在论文列表中选择论文，点击右键菜单或使用批量操作即可使用。

4. Daily arXiv

自动抓取每日最新论文，支持按机构、地区、关键词过滤。感兴趣的论文可一键导入到主库。

5. 从 Zotero 导入

支持从 Zotero 导入论文库。前往 Settings → Import，上传 Zotero 导出的 RDF 文件，系统会自动下载 PDF 并恢复分类结构。
