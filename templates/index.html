Welcome to Resophy

Select AI Output Language

Choose the language for AI-generated content (translation, interpretation, and Daily arXiv summaries):

1. Configure AI features

Click top-right avatar → Agentic to set LLM API and MinerU services. This is required for AI translation, AI analysis, and Daily arXiv.

2. Basic operations

Upload PDF papers, create folders to organize them, and use search to find papers. Click a paper to view details, double-click to open the reader.

3. Agentic Tools

Supports AI analysis and AI translation. AI analysis converts a paper into a detailed Markdown interpretation; AI translation generates bilingual (EN-ZH) PDFs. In the paper list, select papers and use the context menu or batch actions.

4. Daily arXiv

Automatically fetches the latest arXiv papers, with filters by institution, country, and keywords. You can import interesting papers into your main library with one click.

5. Import from Zotero

You can import your Zotero library. Go to top-right avatar → Import, upload the RDF file exported from Zotero, and the system will automatically download PDFs and restore folder structure.
